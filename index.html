☁️ Cloud Deployment Guide

01
AWS S3 Static Hosting

Upload all files to an S3 bucket. Enable "Static website hosting" under bucket properties. Set index.html as root document.

02
GitHub Pages (Free)

Push the folder to a GitHub repo. Go to Settings → Pages → Select branch → Save. Your app is live in seconds.

03
Netlify / Vercel

Drag and drop the project folder into Netlify's dashboard or connect your repo to Vercel for instant deployment.

04
Google Cloud Storage

Create a GCS bucket, upload files, set uniform public access, and add website configuration for index.html.
